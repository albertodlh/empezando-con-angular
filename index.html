Two way data binding (Enlace de datos en dos direcciones)

Angular sincroniza automáticamente los datos entre un modelo y sus componentes de visualización, evitando así inconsistencias de visualización. Cuando los datos de un modelo cambian, la vista reflejará dichos cambios de manera inmediata.
